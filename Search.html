Search Properties

The Search Properties page allows users to easily search and filter real estate listings based on specific criteria. Follow the steps below to effectively use this page:

  1. Accessing the Page

    2. Navigate to the Search Properties page from your application. The page can typically be accessed via a designated route, such as /search.

  2. Sidebar Filters

    3. Search Term: Enter a search term in the designated field to search for properties based on keywords.

    Type: Choose the type of property you are interested in by selecting one of the following options:

    • All
    • Sale
    • Buy
    • Rent

    Amenities:

    • Parking: Check the box to filter properties that include parking amenities.
    • Furnished: Check the box to filter furnished properties.
    • Offer: Check the box to filter properties with active offers.

    Sort: Choose the sorting order of the listings using the dropdown menu. Options include:

    • Price high to low
    • Price low to high
    • Latest
    • Oldest

    Investment Type: Select the investment type from the dropdown menu. Options include:

    • Payment Plan
    • Handover
    • Ready
    • All
  3. Performing the Search

    4. After adjusting the filters based on your preferences, click the "Search" button to initiate the search.

  4. Viewing Results

    5. The page will display real estate listings that match the specified criteria.

  5. Pagination

    6. If the number of listings exceeds the initial display, a "Show more" button will appear at the bottom of the results. Click this button to load additional listings.

  6. Developer Page

    7. To view properties listed by developers, click the "Developer" link in the navigation. This will redirect you to the Real Estate Developer page.

  7. Navigation

    8. Utilize the navigation links to switch between various sections of your real estate application.

By following these steps, users can effectively search and explore real estate listings tailored to their preferences using the Search Properties page.
